# BigData_MovieRecommender

## Front End
### Requirements
- <a href="https://www.npmjs.com/get-npm">NodeJs & NPM</a>
### Specifications
The Front End is made using <a href="https://vuejs.org/index.html">VueJS</a> 
### Getting the website working
1. Open a Terminal
2. Navigate to the Front-End folder
3. Execute the command ```npm install```
4. To run the Website type ```npm run serve```

## Back End
### Requirements
- <a href="https://www.python.org/downloads/">Python</a>
- <a href="https://pip.pypa.io/en/stable/installing/">PIP</a>
- <a href="https://scikit-learn.org/stable/install.html">sklearn</a>
- <a href="https://pypi.org/project/numpy/">numpy</a>
- <a href="https://pypi.org/project/pandas/">pandas</a>
- <a href="https://flask-restful.readthedocs.io/en/latest/installation.html">flask_restful</a>
- <a href="https://pypi.org/project/Flask/">flask</a>

### Getting the server working
1. Open Terminal
2. Navigate to the Back-End folder
3. Execute the command ```python Server.py```
